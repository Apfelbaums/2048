# 2048 - Игра-головоломка

Веб-версия популярной игры 2048 с красивыми анимациями.

## Правила игры

1. **Цель**: Соедините числа, чтобы получить плитку с числом 2048
2. **Управление**: Используйте стрелки на клавиатуре (↑↓←→) для перемещения всех плиток
3. **Механика**: 
   - При каждом ходе все плитки сдвигаются в выбранном направлении
   - Когда две плитки с одинаковыми числами соприкасаются, они объединяются в одну
   - После каждого хода на поле появляется новая плитка (2 или 4)
4. **Победа**: Создайте плитку со значением 2048
5. **Поражение**: Когда на поле нет свободных мест и нельзя объединить соседние плитки

## Особенности

- 🎯 Цель: 2048 (классическая версия)
- 💾 Сохранение лучшего результата в localStorage
- 📱 Адаптивный дизайн (работает на мобильных устройствах)
- 🎨 Плавные анимации с появлением, объединением и движением плиток
- 🔄 Кнопка перезапуска игры

## Как запустить

1. Откройте файл `index.html` в любом современном браузере
2. Или запустите локальный сервер:
   ```bash
   # Если у вас установлен Python 3
   python -m http.server 8000
   
   # Если у вас установлен Node.js
   npx http-server
   ```
3. Откройте http://localhost:8000 в браузере

## Управление

- **Стрелка влево** - Сдвиг всех плиток влево
- **Стрелка вправо** - Сдвиг всех плиток вправо  
- **Стрелка вверх** - Сдвиг всех плиток вверх
- **Стрелка вниз** - Сдвиг всех плиток вниз
- **Кнопка "Новая игра"** - Перезапуск игры
